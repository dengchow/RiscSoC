## 一、简介

本项目是为了模拟世界上最原始的处理器：Intel 4004，使用分立元器件搭建。  

## 二、功能介绍

最多存储 16 条指令，可以进行数字量的输入、输出，以及算术运算。  

## 三、使用说明

- 硬件部分的设计源码存放在 `rtl/` 路径下  

进入 `sim/` 路径下，在各子目录下执行 `make` 即可，将自动完成以下操作  

1. 编译 rtl design  

2. 运行 simulation  

3. 执行结果  

4. 结束  

*执行 make 前请先根据实际情况修改 Makefile 文件*  

控制指令在 `tb/core_tb.v` 文件中进行手动编程。
